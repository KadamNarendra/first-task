# Personal Introduction Program 

This is a simple Python beginner project that asks the user for their
**name**, **age**, and **hobby**, then displays a friendly welcome message.

---

##  Project Files

- personal_intro.py → Main Python program
- requirements.txt → Required Python packages (none needed)
- README.md → Project description and instructions

# Variables – Containers for Data:
A variable stores information so you can use it later.
# Example:
age = 20
name = "Narendra"
is_student = True

# Types of Data:
Data Types      Example      	Meaning
Integer             10	              Whole number
Float	           10.5		Decimal number
String	           "Hello"		Text
Boolean        True / False          Yes or No

 # Basic Math Operations:
Python can do calculations like a calculator.
# Example:
a = 10
b = 5
print(a + b)    # Addition → 15
print(a - b)   # Subtraction → 5
print(a * b)   # Multiplication → 50
print(a / b)   # Division → 2.0

# Strings: – Working with Text
Strings are text values written inside quotes.
# Creating a string:
Example:
name = "Python"
Combining strings (Concatenation):
Ex:
first = "Hello"
second = "World"
print(first + " " + second)
# String operations:
Ex:
text = "python"
print(text.upper())   # PYTHON
print(len(text))      # 6
# Lists: – Collection of Items
A list stores multiple values in one variable.
Example:
numbers = [1, 2, 3, 4]
fruits = ["apple", "banana", "mango"]


# Accessing list items (Index starts from 0):
Ex:
print(fruits[0])  # apple
print(fruits[2])  # mango
Modify a list:
Ex:
fruits.append("orange")
print(fruits)
